# 工具脚本

本目录包含辅助工具脚本，用于开发和测试。

## 📚 文件列表

| 文件                       | 描述                       |
| -------------------------- | -------------------------- |
| `list-models.ts`           | 列出 Gemini 可用的模型列表 |
| `test-function-calling.ts` | Function Calling 功能测试  |

## 🚀 使用方式

```bash
# 查看可用模型
npx tsx src/utils/list-models.ts

# 测试 Function Calling
npx tsx src/utils/test-function-calling.ts
```

## 💡 说明

这些工具脚本不属于学习路线的任何阶段，而是用于：

- 🔍 探索 API 功能
- 🧪 快速测试概念
- 🛠️ 开发调试

你可以根据需要添加更多工具脚本到此目录。
